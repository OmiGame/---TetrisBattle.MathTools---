# 关卡数据生成器

## 功能概述
关卡数据生成器用于根据关卡体验配置限制生成关卡波次配置，确保生成的波次符合以下要求：
1. 所有波次加起来等于最大波次数
2. 相邻两个波次体验尽可能不相同
3. 困难、地狱和Boss波次尽量不相连
4. 关卡内所有波次生成的boss总数不能超过3个
5. 可能生成boss的波次(地狱和boss波)按照关卡体验配置限制中的boss可能出现波次来配置
6. 第一波必须是轻松波次

## 主要类和方法

### 关卡生成器类
- `生成随机关卡配置(关卡ID, 角色平均等级参考, 难度, 适配地图)`: 生成完整的关卡配置
- `生成关卡波次配置(角色平均等级参考, 难度, 适配地图)`: 根据难度和配置限制生成波次配置列表
- `调整波次数量(波次体验数量, 最大波次数)`: 调整各种波次体验的数量，确保总和等于最大波次数
- `创建波次体验序列(波次体验数量, 最大波次数, boss可能出现波次)`: 创建符合要求的波次体验序列
- `调整关卡波次配置(波次配置列表, 难度)`: 调整已生成的波次配置，使其符合要求

## 使用方法

### 生成关卡配置
```python
from LevelData.LevelDataGenerator import 关卡生成器
from LevelData.LevelDataType import 难度类型

# 生成一个普通难度的关卡配置
关卡配置 = 关卡生成器.生成随机关卡配置(
    关卡ID=1,
    角色平均等级参考=10,
    难度=难度类型.普通,
    适配地图="森林"  # 可选参数，默认为"森林"
)

# 打印关卡基础信息
print(f"我方城堡血量: {关卡配置.我方城堡血量}")
print(f"敌方城堡血量: {关卡配置.敌方城堡血量}")
print(f"生成延迟时间: {关卡配置.生成延迟时间}")
print(f"限时生成时长: {关卡配置.限时生成时长}")
print(f"波次数量: {len(关卡配置.波次配置)}")

# 打印每个波次的体验类型
for i, 波次 in enumerate(关卡配置.波次配置):
    print(f"波次{i+1}: {波次.波次体验.value}")
```

### 只生成波次配置
```python
from LevelData.LevelDataGenerator import 关卡生成器
from LevelData.LevelDataType import 难度类型

# 生成一个普通难度的波次配置列表
波次配置列表 = 关卡生成器.生成关卡波次配置(
    角色平均等级参考=10,
    难度=难度类型.普通,
    适配地图="森林"  # 可选参数，默认为"森林"
)

# 打印波次数量
print(f"波次数量: {len(波次配置列表)}")

# 打印每个波次的体验类型
for i, 波次 in enumerate(波次配置列表):
    print(f"波次{i+1}: {波次.波次体验.value}")
```

### 调整波次配置
```python
from LevelData.LevelDataGenerator import 关卡生成器
from LevelData.LevelDataType import 难度类型

# 生成一个普通难度的波次配置列表
波次配置列表 = 关卡生成器.生成关卡波次配置(
    角色平均等级参考=10,
    难度=难度类型.普通,
    适配地图="森林"
)

# 调整波次配置
调整后波次配置列表 = 关卡生成器.调整关卡波次配置(波次配置列表, 难度类型.普通)

# 打印调整前后的波次体验类型
print("调整前:")
for i, 波次 in enumerate(波次配置列表):
    print(f"波次{i+1}: {波次.波次体验.value}")

print("\n调整后:")
for i, 波次 in enumerate(调整后波次配置列表):
    print(f"波次{i+1}: {波次.波次体验.value}")
```

## 配置限制说明
关卡体验配置限制定义在`LevelDataType.py`中，包含以下内容：
- 各种波次体验的数量范围
- boss可能出现波次的范围
- 我方城堡血量
- 敌方城堡血量

每种难度都有对应的配置限制，生成器会根据这些限制生成符合要求的波次配置。
